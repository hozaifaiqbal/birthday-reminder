# 🎂 Birthday Reminder System

[![Python](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![GitHub Actions](https://img.shields.io/badge/GitHub_Actions-Automated-orange.svg)](https://github.com/features/actions)

An automated Python-based notification system that syncs with Google Sheets to ensure you never miss a birthday again.

## 🚀 How It Works
1. **Data Source:** Pulls birthday data from a linked Google Sheet.
2. **Logic:** Python script checks for matches with the current date.
3. **Automation:** Powered by GitHub Actions, running daily at 6:00 AM UTC.

## 🛠️ Tech Stack
* **Language:** Python
* **Data:** Google Sheets API / Pandas
* **Automation:** GitHub Actions
